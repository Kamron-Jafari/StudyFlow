      Google Calendar

      Offer after each block
      When a focus block ends, show an “Add to Google Calendar” button in the banner
      Add last block
      Send your most recent focus block to Google Calendar now
      Export sessions
      Download a .ics of your focus blocks, then import it in Google Calendar
      How this works

      “Add to Google Calendar” opens Google Calendar with the event already filled in — using whatever Google account you're already signed into in your browser. You just review it and hit Save. No API keys, no Google Cloud project, no setup.

      To add many blocks at once, use Export .ics and import the file in Google Calendar ▸ Settings ▸ Import & export. Works online or offline.

      Note: a truly hands-off, click-free background sync isn't possible from a plain web page — Google requires a registered Cloud client ID for that. This approach keeps it zero-setup and uses the sign-in you already have.
